# Magebook editor

Magebook editor for interactive fiction. The perfect tool for writing, editing and shuffling gamebooks.


Discover more at <https://librogamesland.github.io/magebook>

![screenshot](screenshot.png)


## Known Issues

This porting is still in early alpha. Many bugs may occour, and many feature have not been implemented yet.For example, the extension is not integrated with the breadcrumb, nor with the "find"/"replace" navbar commands.

Suggest features or report bugs here: <https://docs.google.com/document/d/1BlAlzgdoFxTNodg501R4kDVG0I7JeBJjPuJsMSP47sg/edit>
